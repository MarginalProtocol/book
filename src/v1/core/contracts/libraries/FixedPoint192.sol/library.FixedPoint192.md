# FixedPoint192
[Git Source](https://github.com/MarginalProtocol/v1-core/blob/4dcf410464dd1b73aaabe9fa06bd3450c672d3b9/contracts/libraries/FixedPoint192.sol)

## State Variables
### RESOLUTION

```solidity
uint8 internal constant RESOLUTION = 192;
```


### Q192
